# calcolatore-fatture
A robust Python script designed to calculate the total cost of items (like photocopies or prints) using tiered pricing (bracket-based calculation). This is the same logic used for utility bills or income tax brackets.
🚀 FeaturesTiered Calculation: Automatically applies different price points based on quantity thresholds.Input Validation: Robust error handling for non-integer inputs and negative numbers.Data Integrity: Uses non-destructive sorting to protect the original reference lists.Currency Formatting: Ensures the output is always displayed with two decimal places (e.g., 9.00€).🛠️ How It WorksThe algorithm uses a "top-down" approach:Normalization: Sorts the reference points to ensure the logic works even if the input lists are unsorted.Bracket Processing: Iterates through the thresholds from highest to lowest.Deduction: If the number of copies exceeds a threshold, the excess is multiplied by that tier's price, and the remaining copies are passed to the next lower tier.Base Case: Any remaining copies (the first bracket) are calculated using the base price.📖 Practical ExampleConfigurationReference Points: [0, 10, 30]Price List: [0.30€, 0.25€, 0.20€]Calculation for 35 Copies:BracketCalculationCostCopies 31 to 35$5 \times 0.20$1.00€Copies 11 to 30$20 \times 0.25$5.00€Copies 1 to 10$10 \times 0.30$3.00€TOTAL9.00€💻 RequirementsPython 3.x🔧 Installation and UsageSave the code into a file named invoice_calc.py.Run the script via terminal:Bashpython invoice_calc.py
