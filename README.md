Python Tiered Invoice Calculator 🧾A robust Python script designed to calculate the total cost of services (such as photocopies or prints) using tiered pricing (bracket-based calculation). This logic mirrors real-world systems used for utility bills, income tax brackets, and bulk-purchase discounts.Features 🎮Tiered Calculation Engine: Automatically applies different price points based on quantity thresholds using a degressive logic.Robust Input Validation: Custom try-except blocks ensure the program doesn't crash on non-numeric strings or invalid inputs.Clean UI Flow: Integrated clear_screen() and time.sleep() utilities to maintain a professional terminal interface and readable error messages.Data Integrity: Uses non-destructive sorting (sorted) to ensure the calculation logic remains accurate even if the reference lists are provided out of order.Currency Formatting: Precision-focused output using f-strings to display two decimal places with the Euro symbol (e.g., 9.00€).Calculation Logic 📋The algorithm uses a "Top-Down" approach to process quantities across different brackets:Normalization: Sorts the reference points to ensure the calculation flow is consistent.Bracket Processing: Iterates through the thresholds from highest to lowest.Deduction: If the quantity exceeds a threshold, the excess is multiplied by that tier's price, and the remaining amount is passed to the next lower tier.Base Case: Any remaining units (the first bracket, e.g., 0-10) are calculated using the base price.Practical Example 🧪Configuration:Reference Points: [0, 10, 30]Price List: [0.30€, 0.25€, 0.20€]Calculation for 35 Copies:BracketCalculationPartial CostUnits 31 to 35$5 \times 0.20€$1.00€Units 11 to 30$20 \times 0.25€$5.00€Units 1 to 10$10 \times 0.30€$3.00€TOTAL9.00€Project Structure 📁Plaintextinvoice-calculator/
